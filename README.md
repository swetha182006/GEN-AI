# AI Resume Analyzer and LinkedIn Scraper using Generative AI

## Introduction

Developed an AI-powered Resume Analyzer and LinkedIn Job Scraper using Streamlit and Generative AI. The application helps users analyze resumes by generating summaries, identifying strengths and weaknesses, and suggesting suitable job roles. It also automates LinkedIn job searching using Selenium to fetch job listings based on user input.

<br />

## Key Technologies and Skills

* Python
* Streamlit
* Pandas
* Selenium
* PyPDF2
* Hugging Face API
* Generative AI
* NLP
* Web Scraping

<br />

## Installation

Install the required packages:

```python
pip install pandas
pip install streamlit
pip install streamlit-option-menu
pip install streamlit-extras
pip install PyPDF2
pip install selenium

```

<br />

## Usage

1. Clone the repository
2. Install dependencies

```python
pip install -r requirements.txt
```

3. Run the application

```python
streamlit run app.py
```

<br />

## Features

### Resume Analysis

* Resume Summary Generation
* Strength Analysis
* Weakness Detection
* Suggested Job Titles

### AI Integration

* Uses Hugging Face Generative AI models for intelligent resume analysis.

### LinkedIn Job Scraper

* Automates LinkedIn job searching using Selenium.
* Extracts:

  * Company Name
  * Job Title
  * Location

### Streamlit Dashboard

* Interactive and user-friendly interface.
* Sidebar navigation for different features.
